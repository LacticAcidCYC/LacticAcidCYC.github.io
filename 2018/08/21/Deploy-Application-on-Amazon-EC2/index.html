Amazon EC2

Elastic Compute Cloud (EC2) : EC2 allows scalable deployment of applications by providing a Web service through which a user can boot an Amazon Machine Image to create a virtual machine, which Amazon calls an “instance”, containing any software desired. A user can create, launch, and terminate server instances as needed, paying by the hour for active servers, hence the term “elastic”.

Step 1. Launch an instance (Ubuntu linux)

  • a. Sign in your aws account and then open EC2 dashboard.
  • b. Launch an instance.
  • c. Select the Ubuntu Server 16.04 image.
  • d. Use t2.micro as instance (free). Click next instead of launch.
  • e. Jump to security group setup. Add ports and set sources (determine the traffic that can reach the instance).
  • f. Click Launch. You will be asked to create and name a new key pair and download the private key.
  • g. Wait for the instance to initialize.
